
### 认证接口

#### request请求

    POST /ihome/auth/
    
##### params参数:

    real_name str 身份证姓名
    
    id_card str 身份证号码
    
#### response响应

##### 失败响应1:

     {
        'code':'902',
        'msg':'信息不能为空'
     }

##### 失败响应2:

    {
        'code':'1007', 
        'msg':'身份证信息错误'
    }
    
##### 失败响应3:

    {
        'code':'900',
        'msg':'数据库访问失败'
    }
    
##### 失败响应4:

    {
        'code':'1004', 
        'msg':'用户名不存在'
    }
    
##### 成功响应:

    {
        'code':'200',
        'msg':'请求成功'
    }
